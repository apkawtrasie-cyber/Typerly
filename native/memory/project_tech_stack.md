Typerly is a **Flutter native** mobile application (not React Native, not web).

**Why:** User confirmed the tech stack is Flutter native.
**How to apply:** All UI code, component suggestions, and design implementations must be written in Dart/Flutter. No HTML, React, or other frameworks. Use Flutter widgets, Material design, and Dart patterns.

Key dependencies observed:
- Flutter + Dart
- flutter_bloc (state management)
- Supabase (backend/database)
- cached_network_image
- flutter_svg
- google_fonts
- Android emulator for testing (Medium_Phone_API)
